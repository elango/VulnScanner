# VulnScanner
This tool will scan a website for vulnerabilities 

Steps to run this tool:
    1. git clone https://github.com/krishpranav/VulnScanner
    2. cd VulnScanner
    3. sudo chmod 777 *
    4. python vulnscan.py    or   sudo ./vulnscan.py
